# WindowsUtilities

Welcome to **WindowsUtilities**! This repository contains a growing collection of utility scripts designed to improve, tweak, and extend your daily Windows user experience.

---

## 🚀 About This Repository

Whether you want to automate tedious tasks, optimize your system, or add quality-of-life improvements, this repository serves as a hub for handy Windows scripts. 

**Key Features:**
* **Optimized for Windows 11:** All scripts are primarily tested, refined, and optimized for Windows 11 environments.
* **Active Development:** This is an ongoing project! New utility scripts and tweaks are added frequently, so be sure to check back or star the repository.
* **UX Focused:** Every script is created with the sole goal of enhancing your workflow and overall user experience.

## 💻 Compatibility

* **Windows 11:** Fully tested and natively optimized.
* **Windows 10:** While tailored for Windows 11, these scripts are written to be backwards compatible and *should* work perfectly fine on Windows 10 as well. 

## ❗ Requirements
* **UpdateAllPrograms:** Winget (on Windows 11 preinstalled)
